# Notice

## Examples

{{% notice success %}}
A success notice
{{% /notice %}}

{{% notice info %}}
An info notice
{{% /notice %}}

{{% notice warning %}}
A warning notice
{{% /notice %}}

{{% notice danger %}}
An danger notice
{{% /notice %}}

## Usage

```
{{%/* notice success */%}}
A success notice
{{%/* /notice */%}}

{{%/* notice info */%}}
An info notice
{{%/* /notice */%}}

{{%/* notice warning */%}}
A warning notice
{{%/* /notice */%}}

{{%/* notice danger */%}}
An danger notice
{{%/* /notice */%}}
```
