# Notice

## Exemplos

{{% notice success %}}
Uma nota de sucesso
{{% /notice %}}

{{% notice info %}}
Uma nota de informações
{{% /notice %}}

{{% notice warning %}}
Uma nota de aviso
{{% /notice %}}

{{% notice danger %}}
Uma nota de perigo
{{% /notice %}}

## Uso

```
{{%/* notice success */%}}
Uma nota de sucesso
{{%/* /notice */%}}

{{%/* notice info */%}}
Uma nota de informações
{{%/* /notice */%}}

{{%/* notice warning */%}}
Uma nota de aviso
{{%/* /notice */%}}

{{%/* notice danger */%}}
Uma nota de perigo
{{%/* /notice */%}}
```
